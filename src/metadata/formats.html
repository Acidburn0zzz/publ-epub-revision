Overview

EPUB 3 defines a small set of core bibliographic metadata that is needed by reading systems in the Package Document in order to provide reliable presentation of an EPUB Publication.

This document provides a guide to metadata record formats commonly linked to from the EPUB Package Document.

note

This guide is only intended to provide an informative list of record formats.

If a format is not listed, it does not imply that it cannot be linked to from the Package Document.
